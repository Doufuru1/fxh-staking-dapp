# FXH 质押挖矿 DAPP

基于海报色调和参考DAPP布局构建的质押和领取收益前端界面。

## 特性

- 🔗 **钱包连接** - 支持Web3钱包连接界面
- 🔒 **质押功能** - 输入质押数量，快速选择比例（25%/50%/75%/MAX）
- 💎 **收益领取** - 显示待领取收益分类（质押挖矿、团队奖励、创世节点分红）
- ⚡ **矿池选择** - 三个矿池：FXH核心池、FXH-ETH LP、创世节点
- 📊 **实时统计** - TVL、累计收益、活跃质押者、区块高度
- 🎨 **暗色主题** - 黑色/炭灰背景配橙色强调色，与海报风格一致

## 设计风格

- **色调**: 深色背景 (#0a0a0f) + 橙色强调 (#ff8c00) + 白色文字
- **字体**: Orbitron (标题) + Noto Sans SC (正文)
- **动效**: 浮动光晕、网格背景、悬停效果、滚动动画
- **布局**: 响应式设计，适配桌面和移动端

## 使用方式

直接在浏览器中打开 `index.html` 文件即可预览。

需要接入智能合约时，请替换 `script` 部分的模拟函数为实际的 Web3 调用。

## 技术栈

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript
